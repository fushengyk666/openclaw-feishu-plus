# Feishu Doc Assistant — 飞书云文档工作流增强

**能力范围**
- 创建文档（指定标题、所在文件夹）
- 读取文档内容（全文 / 纯文本 / 按块分析）
- 文档块操作（创建/更新/删除）
- 文档批量写入工作流

**适用场景**
- 快速创建草稿、周报、会议纪要
- 读取已有文档并做总结、分析、改写
- 模板化生产（从一段描述生成结构化文档）
- 多段内容批量写入文档

**使用指南**

### 创建文档
> "帮我创建一个文档，标题是'周报模板'，里面包含本周工作总结和下周计划"
> "创建一个会议纪要文档"

### 读取文档
> "获取文档 xxx 的内容并总结"
> "把文档 xxx 的纯文本读出来"

### 编辑文档
> "在文档 xxx 中加入一条内容：完成项目 A"

### 批量工作流
> "把以下三条会议记录分别写成段落，追加到文档 xxx 中"
> "读取文档 xxx，提取所有待办事项，整理成列表"

**身份策略**
- 默认：有用户授权时用用户身份操作文档（看到的是用户自己的文档权限），否则用应用身份
- 创建文档在用户身份下，文档所有者为用户本人

**工具列表**
| 工具名 | 说明 |
|--------|------|
| `feishu_plus_doc_create` | 创建文档 |
| `feishu_plus_doc_get` | 获取文档信息 |
| `feishu_plus_doc_list_blocks` | 列出文档块 |
| `feishu_plus_doc_raw_content` | 获取文档纯文本 |

**注意事项**
- 飞书文档使用 `document_id` 标识，不同于文件 `token`
- 大文档可能需要分页获取块列表
- 块操作需要先获取 `parent_block_id`（通常为文档根块）
